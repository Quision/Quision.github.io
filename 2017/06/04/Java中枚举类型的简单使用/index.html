Java中枚举类型的简单使用

Jun 4, 2017 | Hits

1、概述

　　在Java语言中有一个“enum”关键字，它可以将一组具名的有限集合创建为一种新的类型，而这些具名的值可以作为常规程序组件使用，这是一种非常有用的功能。

2、enum的特性

　　enum基本特性如下：

  1. values()方法，可以遍历enum实例，返回enum实例的数组；
  2. ordinal()方法，返回一个int值，每个enum实例在声明时的次序，从0开始；
  3. 对比，可以使用“==”来对比enum实例，编译器自动提供equals()和hashCode()方法；
  4. Enum类实现了Comparable方法，所以实例具有compareTo方法；
  5. name()方法，返回实例声明时的名字，与使用toString()方法效果一直，valueOf()是在Enum中定义的static方法，给定特定的名字，会返相应的实例，如果不存在则抛出异常；
  6. enum类不可继承，除此之外，我们可以把enum当作普通的类对待。
